## Description

CogsSDK Example Application

## Requirements

* iOS 9.0+
* Xcode 8.0+
* Swift 3.0+

## Usage

* Navigate with the command line to the folder where ``Podfile`` is situated
* Execute the following command in the terminal:

```bash
$ pod install
```
* Open `CogsSDKExampleApp.xcworkspace`
* Run
